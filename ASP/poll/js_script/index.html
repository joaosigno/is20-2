Tigra Calendar
Description

Tigra Calendar - Popup timestamp picker is JavaScript applet that makes it easy for your visitors to fill out the date/time field(s) of your form, by selecting the date from a popup calendar.

Script allows to save the time for those internet/intranet developers who need intuitive, easy to use client-side timestamp input control.

Frequently Asked Questions
1. What browsers are supported?
Shortly: Any browsers capable of running generic JavaScript.
Script has been actually tested on different versions of Microsoft Internet Explorer, Netscape Navigator/Communicator, and Opera.
2. Is there any way to change the format of the date in the Tigra Calendar?
There is no built-in parameter driven way to change date/time format. In most cases you only need one particular format to be supported, so all the code for other formats is just the waste of computer and network resource.
Tigra Calendar is available free with European and American time/date formats. If you need some other then modify datetime parsing/generating functions at the end of the script file (see the comments in script source). If you can't do this yourself then ask someone else who can. Finally you can purchase script customization from me which includes colors scheme tuning for your application.
