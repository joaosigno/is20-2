PRODUCT VERSION

This Active Server Pages (ASP) sample application is for use with version 8.5 of Crystal Reports (build 8.5.0.217).

________________________________________________________________ 

DESCRIPTION

This sample application demonstrates using the Report Designer Component (RDC) to display a report that contains a selection formula.

INSTALLATION

1. Extract the contents of the zip file into a new folder.

2. Create any ODBC System DSNs required by the report.

3. Map a virtual directory in the Internet Service Manager to this new folder.

4. Make sure that this new directory has Read and Execute permissions. You may also want to turn on Directory Browsing.

5. Browse to this virtual directory and call SimpleSelectionFormula.asp

==================
NOTE:

If you wish to use a different Report Viewer, you simply need to change the #include statement in SimpleSelectionFormula.asp which specifies the Report Viewer used.
